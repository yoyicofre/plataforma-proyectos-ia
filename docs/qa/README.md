# QA

Este directorio contiene la estrategia de testing y planes por módulo/cambio.

## Convenciones
- **Bugfix**: siempre agrega test de regresión.
- **Feature**: unit tests + integración del endpoint principal.
- **Módulo nuevo**: al menos 1 test de smoke (import + endpoint base).

## Archivos
- `test-strategy.md`: estrategia general.
- `test-plan-template.md`: plantilla para plan por cambio/módulo.
